## Contributing

Thanks for helping improve **TikTok LIVE Watcher**.

### Dev setup

- Install Node.js (LTS recommended)
- In the project folder:

```bash
npm install
npm start
```

### What to include in PRs

- Clear description + screenshots (if UI)
- Steps to test
- Keep changes focused (one feature/fix per PR if possible)

### Notes

- This app uses a live status service for LIVE status + events.
- Please avoid adding features that require many parallel socket connections (rate limits).
