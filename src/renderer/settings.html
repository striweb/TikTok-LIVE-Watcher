Профили и настройки
Zerody backend
Профили
Добавяне, изтриване и редактиране на usernames + per-host интервал (по подразбиране = global).
Настройки
Интервал за проверка + параметри за Zerody overlay.
Минимум 1 минута.
Ползва се за “Отвори” и при клик на нотификация.
Анти-спам: ще ограничи повторните известия за същия viewer/host.
0 = без cooldown (не препоръчвам).
Автоматично стартира “Всички LIVE” във фонов режим, когато има LIVE профили.
Ще записва “gift_sent” събития (кой, в кой лайф, час).
По подразбиране е изключено (може да спами при активни чатове).
Anti-spam за известията при gifts (0 = без cooldown).
Пуска звук при LIVE start и при viewer_joined.
Ще промени Theme/Accent/Density.
Kanban показва профилите като колони (LIVE/Offline/Unknown).
Maintenance
Рестарт и изчистване на cache (помага при “забили” ресурси/стари UI версии).
Share config
Export/Import на настройки + списъци (usernames + watched viewers).
